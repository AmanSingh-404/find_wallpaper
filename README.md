# PixelVerse - Find Your Perfect Wallpaper

PixelVerse is a modern, responsive web application for discovering high-quality photos, videos, and collections. Powered by the Unsplash and Pexels APIs, it offers a stunning, immersive interface for finding the perfect visuals for your next project.

![PixelVerse Demo](https://images.unsplash.com/photo-1451187580459-43490279c0fa?q=80&w=2072&auto=format&fit=crop)

## Features

*   **Stunning UI/UX**: A modern, dark-themed interface with glassmorphism effects, smooth animations, and a responsive layout.
*   **Search Functionality**: Powerful search across photos, videos, and collections.
*   **Media Types**: Support for high-resolution images and videos.
*   **Collections**: Browse curated collections of diverse themes.
*   **Favorites**: Save your favorite media for quick access later.
*   **Infinite Scroll**: Seamlessly load more content as you explore.
*   **Modal View**: Detailed view for media items.

## Technologies Used

*   **React**: Frontend library for building the user interface.
*   **Redux Toolkit**: For efficient state management.
*   **React Router**: For handling navigation and routing.
*   **Tailwind CSS**: For rapid, utility-first styling.
*   **Lucide React**: For beautiful, consistent icons.
*   **Vite**: For fast development and optimized builds.
*   **Axios**: For making API requests.

## Getting Started

### Prerequisites

*   Node.js (v14 or higher)
*   npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/AmanSingh-404/find_wallpaper.git
    cd find_wallpaper
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Environment Setup:**
    Create a `.env` file in the root directory and add your API keys:
    ```env
    VITE_UNSPLASH_API_KEY=your_unsplash_api_key
    VITE_PEXELS_API_KEY=your_pexels_api_key
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```

5.  **Build for production:**
    ```bash
    npm run build
    ```
